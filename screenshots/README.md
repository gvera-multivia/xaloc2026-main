# Screenshots

Este directorio contiene las capturas de pantalla generadas durante la ejecución.

## Tipos de Screenshots

### Screenshots de Éxito
- **Formato:** `xaloc_final_YYYYMMDD_HHMMSS.png`
- **Contenido:** Pantalla final de confirmación antes de enviar

### Screenshots de Error
- **Formato:** `error.png`
- **Contenido:** Pantalla donde ocurrió el error

## Notas

- Las capturas se toman en formato PNG
- Se usa `full_page=True` para capturar toda la página
- Los archivos se sobrescriben en cada ejecución (error.png)
