# Xaloc Automation - Logs

Este directorio contiene los archivos de log de las ejecuciones del bot.

## Archivos Generados

- `xaloc_automation.log` - Log principal con todas las operaciones

## Formato de Log

```
YYYY-MM-DD HH:MM:SS - LEVEL - MESSAGE
```

## Niveles de Log

- **INFO:** Operaciones normales
- **WARNING:** Advertencias (ej: modo testing activado)
- **ERROR:** Errores capturados
